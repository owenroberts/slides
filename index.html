This is slides

Slides is a little program I made to host notes for classes that can toggle between a slides/power point style view and regular blog/outline style view, using HTML/CSS/JavaScript so it can live online.

Click the button in the upper right corner to make this look like a slides presentation. Or just hit the s key.

Click the button to toggle back to outline. Or hit the o key.

In slides mode, use arrow keys K and L for next slide, J and I for previous slide.
